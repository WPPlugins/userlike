== Description ==

This plugin integrates the [Userlike](https://www.userlike.com/ "Live Chat for Website Support") Live Chat into your website.

Userlike is a live chat software with which you can chat with the visitors of your website. Offer an outstanding online service, get to know your visitors, and raise your conversion rate.  For those serious about customer support. Used by 19000+ companies worldwide. Help when it matters, where it matters.
This plugin allows you to embed Userlike into your WordPress website within seconds. Please be aware that you need an account with Userlike to use this plugin. 

== Installation ==

1. Extract `userlike.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to 'Settings' -> 'Userlike' and enter your Userlike Widget Secret, which can be found in the Userlike Widget Editor in the [install tab](https://www.userlike.com/en/dashboard/config/chat_widget/edit/default#install-credentials).
